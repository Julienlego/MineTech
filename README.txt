--- Tips for GitBash ---
TO PULL:
(remove quotes)
1.Right click on repo folder, "Git Bash Here"
2. "git init" in shell.
3. "git pull origin master" in shell

TO PUSH:
1. 1 and 2 above
2. "git add <filename>" for each new file (usually just minetech)
3. "git commit -m <message>" with a commit message replacing <message>
4. "git push -u origin master"
